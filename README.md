## üìã Overview

Tableau_European_Spending is a ready-to-use dashboard application. It shows easy to understand visual summaries of European Union consumption trends. The data comes from Eurostat, cleaned and combined for clarity. This project brings insights on EU spending, covering many household categories using COICOP classification. The interactive dashboard lets you explore these trends visually without needing any technical skills.

The tool focuses on clear, interactive charts. It uses calculated fields and relational joins behind the scenes to show accurate insights. Whether you want to study macroeconomic patterns or understand specific consumption habits, this dashboard gives you instant access.
